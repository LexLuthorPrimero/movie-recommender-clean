# 🎬 Movie Recommender

Sistema de recomendación de películas basado en embeddings de géneros y ratings promedio, con similitud por coseno.

## Tecnologías

- Python 3.13
- Streamlit
- Pandas, NumPy

## Uso local

```bash
git clone https://github.com/LexLuthorPrimero/movie-recommender-clean.git
cd movie-recommender-clean
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
```
